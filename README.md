## 轮子api

### button

| 属性         | 说明                                       | 类型    | 默认值 |
| ------------ | ------------------------------------------ | ------- | ------ |
| iconName     | 按钮添加icon                               | string  | -      |
| iconPosition | icon位置, 只能在left和right选择            | string  | -      |
| loading      | 点击之后, 按钮不可点击, 一定条件下才能点击 | boolean |        |
|              |                                            |         |        |

### icon

| 属性     | 说明         | 类型   | 默认值 |
| -------- | ------------ | ------ | ------ |
