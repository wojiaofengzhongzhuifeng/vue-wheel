    # grid 布局

    以 24 份宽度为标准，快速创建布局

    # 基本使用

    <w-row gutter="20px">
    <w-col :span="4">
        <div style="background: #d3dce6;text-align: center">
            1
        </div>
    </w-col>
    <w-col :span="15" offset="5">
        <div style="background: #e5e9f2;text-align: center">
            2
        </div>
    </w-col>
</w-row>

    # 响应式布局

    <w-row gutter="20px">
    <w-col :span="4" :phone="{span: 8}">
        <div style="background: #d3dce6;text-align: center">
            1
        </div>
    </w-col>
    <w-col :span="15" offset="5"  :phone="{span: 8, offset: 8}">
        <div style="background: #e5e9f2;text-align: center">
            2
        </div>
    </w-col>
</w-row>

    # row Attributes

    参数 说明 类型 可选值 默认值
    gutter row组件中相邻col组件的间隔 String - -

    # col Attributes

    参数 说明 类型 可选值 默认值
    span col占row的份数 Number - -
    offset col组件左侧透明份数 Number - -
    phone <576px屏幕宽度 Object span / offset -
    ipad <768px屏幕宽度 Object span / offset -
    narrowPc <992px屏幕宽度 Object span / offset -
    pc <1200px屏幕宽度 Object span / offset -
    widthPc <1600px屏幕宽度 Object span / offset -
    widthWidthPc >1600px屏幕宽度 Object span / offset -
