    # input 文本输入框

    代替原生 input，并且具有更多功能

    # 带有 icon 的input

    <w-input :icon="{name: 'setting', message:'设置'}"></w-input>

    # Attributes

    参数 说明 类型 可选值 默认值
    icon input 的 icon Object name / message -
    value input 的 展示内容 String / Number - -
    disabled input 是否可输入 Boolean true / false true

    # 其他

    可以在 input 组件中@click=xxx监听点击事件，包括以下事件监听：change ，input ，focus ， blur ， click
